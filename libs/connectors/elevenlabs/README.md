# genblaze-elevenlabs

ElevenLabs TTS and sound-effects provider adapter for [genblaze](https://github.com/backblaze-labs/genblaze).

## Install

```bash
pip install genblaze-elevenlabs
```

Registers `elevenlabs-tts` and `elevenlabs-sfx` providers via entry points; `genblaze-core` discovers them automatically.

## Documentation

Voice catalog, credentials, and examples live in the monorepo: https://github.com/backblaze-labs/genblaze
