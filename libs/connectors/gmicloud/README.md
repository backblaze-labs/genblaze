# genblaze-gmicloud

GMICloud media provider adapters for [genblaze](https://github.com/backblaze-labs/genblaze) — video, image, and audio models served by GMICloud.

## Install

```bash
pip install genblaze-gmicloud
```

Registers `gmicloud`, `gmicloud-image`, and `gmicloud-audio` providers via entry points; `genblaze-core` discovers them automatically.

## Documentation

Model catalog, credentials, and examples live in the monorepo: https://github.com/backblaze-labs/genblaze
