# genblaze-openai

OpenAI provider adapters for [genblaze](https://github.com/backblaze-labs/genblaze) — Sora (video), DALL·E (image), and TTS (audio).

## Install

```bash
pip install genblaze-openai
```

Registers `openai-sora`, `openai-dalle`, and `openai-tts` providers via entry points; `genblaze-core` discovers them automatically.

## Documentation

Model selection, credentials, and examples live in the monorepo: https://github.com/backblaze-labs/genblaze
