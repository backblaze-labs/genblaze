# genblaze-decart

Decart Lucy video and image provider adapter for [genblaze](https://github.com/backblaze-labs/genblaze).

## Install

```bash
pip install genblaze-decart
```

Registers `decart` (video) and `decart-image` providers via entry points; `genblaze-core` discovers them automatically.

## Documentation

Model selection, credentials, and examples live in the monorepo: https://github.com/backblaze-labs/genblaze
