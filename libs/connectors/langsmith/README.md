# genblaze-langsmith

LangSmith tracer backend for [genblaze](https://github.com/backblaze-labs/genblaze). Forwards pipeline spans, provider calls, and manifest events to a LangSmith project for observability.

## Install

```bash
pip install genblaze-langsmith
```

## Documentation

Configuration, environment variables, and examples live in the monorepo: https://github.com/backblaze-labs/genblaze
