# genblaze-lmnt

LMNT text-to-speech provider adapter for [genblaze](https://github.com/backblaze-labs/genblaze).

## Install

```bash
pip install genblaze-lmnt
```

Registers the `lmnt` provider via entry points; `genblaze-core` discovers it automatically.

## Documentation

Voice catalog, credentials, and examples live in the monorepo: https://github.com/backblaze-labs/genblaze
