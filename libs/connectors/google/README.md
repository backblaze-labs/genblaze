# genblaze-google

Google provider adapters for [genblaze](https://github.com/backblaze-labs/genblaze) — Veo (video) and Imagen (image).

## Install

```bash
pip install genblaze-google
```

Registers `google-veo` and `google-imagen` providers via entry points; `genblaze-core` discovers them automatically.

## Documentation

Authentication, model selection, and examples live in the monorepo: https://github.com/backblaze-labs/genblaze
