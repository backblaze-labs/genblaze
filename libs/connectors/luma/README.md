# genblaze-luma

Luma Dream Machine video provider adapter for [genblaze](https://github.com/backblaze-labs/genblaze).

## Install

```bash
pip install genblaze-luma
```

Registers the `luma` provider via entry points; `genblaze-core` discovers it automatically.

## Documentation

Model selection, credentials, and examples live in the monorepo: https://github.com/backblaze-labs/genblaze
