# genblaze-cli

Command-line interface for [genblaze](https://github.com/backblaze-labs/genblaze) manifest operations — verify, inspect, and audit provenance manifests produced by genblaze pipelines.

## Install

```bash
pip install genblaze-cli
```

This installs the `genblaze` console script.

## Usage

```bash
genblaze --help
```

Full documentation lives in the monorepo: https://github.com/backblaze-labs/genblaze
